# Daily Tarot Fortune

每日塔罗运势网站，结合十二星座的塔罗抽牌应用。

## 功能特性

- ✨ 梦幻薰衣草紫色主题
- ♈ 十二星座选择
- 🃏 78张韦特塔罗牌
- 📖 结合星座的运势解读
- ✨ 精美的视觉效果

## 技术栈

- React 18 + TypeScript
- Vite 6
- Tailwind CSS 3
- Zustand (状态管理)
- Framer Motion (动画)
- React Router (路由)

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 项目结构

```
src/
├── components/       # 组件
├── pages/           # 页面
├── data/            # 数据
├── stores/          # 状态管理
├── utils/           # 工具函数
├── images/          # 图片资源
└── fonts/           # 字体文件
```

## 部署

项目已配置为支持 Vercel 部署。

### Vercel 部署步骤

1. 在 GitHub 上创建仓库
2. 将代码推送到仓库
3. 在 Vercel 中导入项目
4. Vercel 会自动检测并配置构建

### 构建配置

- **构建命令**: `npm run build`
- **输出目录**: `dist`
